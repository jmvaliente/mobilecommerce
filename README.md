# mobilecommerce

_App Mobile Ecommerce_

## Starter 🚀

### Pre-requisitos 📋

_install dependencies_

```
npm install
```

### Install 🔧

_Create Build_

```
npm run build
```

_Mode Dev_

```
npm run start
```

_Check Lint_

```
npm run lint
```

_Launch Test_

```
npm run test
```

## Deploy 📦
